Principais formatações

Negrito / Desaques

Nessa frase temos um termo em negrito usando a tag "b" (não semântica).

Nesta frase, tenho um termo em destaque usando o "strong" (que no caso ele realmente destaca a importância de um modo semântico).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag "i" (não semântica).

Nesta frase, temos um termo em ênfase usando a tag "em" (semântica).

Texto Marcado

Podemos criar também um texto marcado usando a tag "mark". E caso eu queira mudar a cor do marca texto eu devo usar uma formatação de CSS no "mark" utilizando o "style".

Texto grande e pequeno

Estamos criando um texto grande ( "big" é de forma não semântica) e outro pequeno neste parágrafo;

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Texto sobrescrito / texto subscrito

Sobrescrito para enserir coisas elevadas no texto. Ex: x20+3
E subscrito para valores ou texto abaixo. EX: H20
